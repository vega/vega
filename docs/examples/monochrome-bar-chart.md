A bar chart using the built-in [`"monochrome"` pattern scheme](../../docs/schemes/#patterns), a print-friendly palette that interleaves solid greyscale values with [pattern](../../docs/types/#Pattern) fills - for figures that must remain legible without color, such as grayscale printing. Legend swatches render the patterns automatically.
